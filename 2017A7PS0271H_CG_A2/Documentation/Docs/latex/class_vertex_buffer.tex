\doxysubsection{Detailed Description}
A wrapper class around the Open\+GL gl\+Vertex\+Buffer type.

Earlier version would delete the buffer when the desstructor was called, meaning that subsequent draw calls would need to reload the buffer from C\+PU memory The current version does not handle G\+PU memory management,but is much faster, so use accordingly
