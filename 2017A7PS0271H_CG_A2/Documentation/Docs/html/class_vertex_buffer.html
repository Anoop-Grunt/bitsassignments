Detailed Description

A wrapper class around the OpenGL glVertexBuffer type.

Earlier version would delete the buffer when the desstructor was called, meaning that subsequent draw calls would need to reload the buffer from CPU memory The current version does not handle GPU memory management,but is much faster, so use accordingly
